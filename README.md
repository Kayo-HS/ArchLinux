## Instalação

Clone o repositório:

```bash
git clone https://github.com/Kayo-HS/ArchLinux
```

Copie os arquivos para sua pasta `.config`:

```bash
cp -r hypr ~/.config/
cp -r waybar ~/.config/
cp -r wofi ~/.config/
cp -r foot ~/.config/
```
